## 技术栈
- TensorFlow.js（前端AI模型）
- Chart.js（数据可视化）
- 纯HTML/CSS/JavaScript，无需后端

## 可调参数说明
- 训练轮数：控制模型训练迭代次数，数值越大拟合效果越好，训练时间越长
- 时间步长：控制模型参考的历史数据长度，影响预测的时序关联性
- 预测长度：控制模型输出的未来数据点数，可自定义预测范围
